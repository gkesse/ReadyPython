# Variable

Les **Variables** en Python se déclarent sans la spécification du Type.  
Python est un langage assez évoluer pour auto-identifier le type de la variable manipulée.

# Programme de Test

```
#================================================
m_int = 1234
m_double = 12.34
m_string = "Hello World !"
m_format = ""

m_format += "m_int : " + str(m_int) + "\n"
m_format += "m_double : " + str(m_double) + "\n"
m_format += "m_string : " + m_string + "\n"

print(m_format)
#================================================
```
