# ReadyPython

**ReadyPython** est une bibliothèque de fonctions Python pour le développement d'applications en Python.  
C'est une bibliothèque développée en Python et maintenu par 
[Gerard KESSE](http://5.48.215.193:8855/presentation/ "Accédez à mon site web (ReadyDev)").

# Technologies

Le développement de la bibliothèque **ReadyPython** a nécessité les technologies suivantes :

* Langage : **Python**
* Editeur de Texte : **Notepad++**
* Outil de Déploiement : **CxFreeze**
* Cible : **Windows, UNIX**
